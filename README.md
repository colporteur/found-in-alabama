# Found in Alabama

The website for Found in Alabama — a reseller of estate finds, vintage, books, ephemera, and small antiques across six marketplaces.

## Local development

```bash
npm install
npm run dev
```

Then open http://localhost:3000.

## Stack

- Next.js 14 (App Router) with TypeScript
- Tailwind CSS for styling
- Deployed on Vercel
- Domain: foundinalabama.com

## Phase 1 (current)

Static site with About / We Buy / Contact / Find Me pages, brand identity matching the business card, and footer linking to all marketplace and social profiles.

## Phase 2 (next)

- Posts system: haul stories, live sale promos, travel/picker appointments
- Chrome extension to capture newly listed items from Nifty.ai
- Staging interface to sort items into haul posts
- Claude API integration for AI-generated narrative drafts
- Sale detection via nightly Nifty CSV export
